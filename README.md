# obesity_phenotypes_falls

This folder contains analysis code developed as part of the **Non-Fracture Fall Injuries and Long-Term Geriatric Consequences Cross-Cohort Study**.

### GEE Analyses: Obesity Phenotypes and Falls

This folder contains R and SAS code for GEE analyses examining associations between obesity phenotypes (whole-body, abdominal, sarcopenic obesity) and fall outcomes (incident falls, recurrent falls, fall counts).
The codes include data setup, model specification, covariate adjustment, and generation of analytic outputs for longitudinal fall analyses.
